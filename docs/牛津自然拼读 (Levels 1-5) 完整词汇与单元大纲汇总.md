# **牛津自然拼读 (Oxford Phonics World) 1-5级全套词汇大纲汇总**

本文档按照从小到大的级别（Level 1 \- Level 5）以及单元顺序（Unit 1 \- Unit 8），全面汇总了《牛津自然拼读》系列的核心词汇与发音规则，为您开发系列塔防游戏提供完整的数据支撑和进阶参考。

## **Level 1: The Alphabet (字母与基础发音)**

第一级别主要聚焦26个英文字母的Letter Name（字母名）和Letter Sound（字母音），培养最基础的语感。

| 单元 (Unit) | 目标字母/发音 | 核心词汇示例 |
| :---- | :---- | :---- |
| **Unit 1** | Aa, Bb, Cc | apple, ant, alligator; bear, bird, bed; cat, cup, car |
| **Unit 2** | Dd, Ee, Ff | dog, desk, duck; egg, elephant, elbow; fish, fan, fork |
| **Unit 3** | Gg, Hh, Ii | goat, gorilla, gift; hat, horse, house; igloo, ink, iguana |
| **Unit 4** | Jj, Kk, Ll | jam, jet, juice; kangaroo, key, kite; lion, lamp, leaf |
| **Unit 5** | Mm, Nn, Oo | monkey, milk, mouse; nut, net, nest; octopus, ox, ostrich |
| **Unit 6** | Pp, Qq, Rr | panda, pen, pig; queen, quilt, question; rabbit, rose, rice |
| **Unit 7** | Ss, Tt, Uu, Vv | sun, seal, sock; tiger, turtle, tent; umbrella, up, uncle; van, vest, vet |
| **Unit 8** | Ww, Xx, Yy, Zz | wolf, web, water; fox, box, six (x作为词尾); yo-yo, yak, yogurt; zebra, zoo, zero |

## **Level 2: Short Vowels (短元音与CVC拼读)**

第二级别进入CVC（辅音-元音-辅音）单词的拼读，掌握5个短元音 a, e, i, o, u 的家族词汇 (Word Families)。

| 单元 (Unit) | 拼读家族 | 核心词汇示例 |
| :---- | :---- | :---- |
| **Unit 1** | Short a (am, an) | ram, jam, yam, dam; pan, man, fan, can |
| **Unit 2** | Short a (ad, ag, ap, at) | dad, sad; bag, rag; cap, map; cat, bat, mat |
| **Unit 3** | Short e (ed, eg, en, et) | bed, red; peg, leg; pen, ten, hen; pet, net, wet |
| **Unit 4** | Short i (ig, in, ip, it) | pig, wig, dig; pin, bin, fin; lip, zip, dip; sit, hit, pit |
| **Unit 5** | Short o (og, op, ot, ox) | dog, log, jog; mop, hop, pop; pot, hot, dot; box, fox, ox |
| **Unit 6** | Short u (ub, ug, up, ut) | tub, cub, sub; bug, mug, rug; cup, pup; nut, hut, cut |
| **Unit 7** | Review & Extension 1 | 综合复习短元音 a, e, i 的CVC词汇，增强分辨能力。 |
| **Unit 8** | Review & Extension 2 | 综合复习短元音 o, u 的CVC词汇，进行整句拼读启蒙。 |

## **Level 3: Long Vowels (长元音与Magic E)**

第三级别重点在于“神奇的e (Magic e)”规则以及常见元音组合，让元音发其字母本身的音。

| 单元 (Unit) | 长元音规则 | 核心词汇示例 |
| :---- | :---- | :---- |
| **Unit 1** | Long a (a\_e) | cake, bake; name, game; gate, late; cave, wave |
| **Unit 2** | Long i (i\_e) | kite, bite; bike, hike; five, dive; time, dime |
| **Unit 3** | Long o (o\_e) | bone, cone; rope, hope; home, dome; rose, nose |
| **Unit 4** | Long u (u\_e) | cube, tube; mule, rule; cute, mute |
| **Unit 5** | ee, ea (Long e) | bee, tree, green; leaf, meat, read, sea |
| **Unit 6** | ai, ay (Long a) | rain, train, tail; day, play, say, way |
| **Unit 7** | oa, ow (Long o) | boat, coat, soap; bow, row, yellow, snow |
| **Unit 8** | igh, y (Long i) | night, light, high; fly, sky, cry, my |

## **Level 4: Consonant Blends (辅音连读)**

第四级别攻克辅音连缀现象，包含两个或多个辅音在一起的发音技巧。

| 单元 (Unit) | 辅音连读组合 | 核心词汇示例 |
| :---- | :---- | :---- |
| **Unit 1** | L-Blends (bl, cl, fl, gl, pl, sl) | black, blanket; clock, club; flag, fly; glass, globe; plant, plane; slide, sleep |
| **Unit 2** | R-Blends (br, cr, fr, gr, pr, tr) | bread, bride; crab, cross; frog, frame; green, grass; prize, prop; tree, truck |
| **Unit 3** | S-Blends (sm, sn, sp, sw, st) | smile, smoke; snow, snake; spoon, spot; swim, swing; star, stop |
| **Unit 4** | Digraphs 1 (sh, ch, tch, wh, ph) | ship, shell; chick, lunch; catch, watch; whale, white; dolphin, photo |
| **Unit 5** | Digraphs 2 (th, ck, qu) | three, teeth, mother, bath; duck, clock, black; queen, quilt, quick |
| **Unit 6** | Ending Blends 1 (ng, nk, nd, nt) | ring, king, sing; bank, pink, tank; hand, sand, wind; tent, ant, plant |
| **Unit 7** | Ending Blends 2 (mp, sk, lt) | camp, lamp, jump; desk, mask, disk; belt, melt, adult |
| **Unit 8** | Review & Soft C/G | city, mice (soft c); cage, page, giraffe (soft g) |

## **Level 5: Letter Combinations (字母组合与进阶)**

第五级别为自然拼读的最高阶，涵盖特殊的元音控制发音、双元音、弱读音及常见词缀。

| 单元 (Unit) | 特殊发音组合 | 核心词汇示例 |
| :---- | :---- | :---- |
| **Unit 1** | R-controlled Vowels (ar, ir, ur, er, or) | car, star, park; bird, girl; nurse, purple; teacher, sister; doctor, tractor |
| **Unit 2** | Diphthongs (ou, ow, oi, oy, oo) | mouse, house; cow, owl; coin, boil; boy, toy; book, pool; push, pull |
| **Unit 3** | Special Combinations (au, aw, all, wa) | August, auto; draw, straw; ball, tall; water, walk |
| **Unit 4** | R-Vowel Advanced (are, air, ea, ear) | bear, pear; hair, chair; head, bread; clear, ear, hear |
| **Unit 5** | Open Syllables (a, e, i, o, u) | baby, lady; he, she; tiger, spider; hotel, cold; music, uniform |
| **Unit 6** | Schwa (弱化音) | panda, banana; chicken, pencil; lemon, monkey; son, love (特殊o) |
| **Unit 7** | Silent Letters (kn, wr, mb, rh, st) | knee, knife; write, wrong; lamb, comb; rhino; castle, whistle |
| **Unit 8** | Suffixes (词缀: tion, sion, ture, ous) | station, action; television, excursion; picture, nature; famous, dangerous |
