# 《拼读保卫战》用户使用手册

> 适用版本：v0.1.0 原型版
> 适用人群：家长 / 老师引导 4-8 岁儿童体验

---

## 一、游戏简介

《拼读保卫战》是一款结合**自然拼读（Phonics）**与**塔防**玩法的教育小游戏。
孩子通过拖拽字母块和词根块，拼出单词，召唤出守卫基地的**单词英雄**！

- 🔵 **蓝色块** = 辅音字母（c / b / p / d…）
- 🟢 **绿色块** = 词根块（-at / -ig / -en…）
- 🟡 **金色格** = 合成成功的单词英雄（cat 🐱 / pig 🐷 / dog 🐶…）

## 三、基本操作

### 3.1 把字母块放入战场

1. 用手指（或鼠标）**长按并拖动**手牌区的字母块
2. 拖到战场格子上方后**松开手指**
3. 字母块落入格子，显示在战场中

> 💡 空格子在拖动经过时会发光提示

### 3.2 合成单词英雄（核心玩法！）

**方法一：手牌拖到格子**
1. 先把辅音字母（如 **c**）拖入一个空格子
2. 再把词根块（如 **-at**）**拖到 c 所在的格子**上
3. 如果能拼成单词 → 格子变成金色英雄 🐱 **CAT**！

**方法二：格子拖到格子**
1. 把战场中的字母块拖到另一个有词根的格子上
2. 同样触发合成判断

> 如果两个块无法合成单词，系统会自动交换它们的位置。

### 3.3 把字母退回手牌

- **双击**战场中的字母块 → 字母回到手牌区
- 注意：已合成的英雄（金色格子）**不能退回**

### 3.4 在格子间移动字母

- 直接拖动战场中的字母块到另一个**空格子** → 字母移动

### 3.5 重新开始

- 点击底部「**重新开始**」按钮 → 战场清空、手牌恢复初始状态

## 六、家长 / 教师使用建议

### 引导话术

1. **认字母**：「这个蓝色块写的是什么字母？它发什么音？」
2. **认词根**：「这个绿色块 -at 是一个'音块'，它发 /æt/ 的音。」
3. **拼读合成**：「把 c 和 -at 放在一起，/k/ + /æt/ = /kæt/，是什么单词？」
4. **看 emoji 猜意思**：「🐱 是什么动物？对，是猫，英文叫 cat！」
5. **升级挑战**：「你能找到所有能和 -ig 合成的字母吗？」

### 建议游戏顺序

| 步骤 | 目标 |
|------|------|
| 1 | 先让孩子随意拖动，熟悉操作 |
| 2 | 家长指定一个目标词（如 cat），引导孩子找到正确字母 |
| 3 | 孩子自己尝试合成所有能和某个词根（如 -at）组合的单词 |
| 4 | 看英雄 emoji，让孩子说出中文意思，加深记忆 |
